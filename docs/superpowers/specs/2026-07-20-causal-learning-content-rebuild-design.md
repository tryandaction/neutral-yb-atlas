# Neutral Yb Atlas 因果学习内容重构设计

## 1. 目标与读者

本轮把网站重构为面向“具备大学物理与线性代数基础、刚进入量子计算”的系统学习图谱。正文必须从计算的基本对象出发，连续回答以下问题：

1. 什么是可验证的计算？
2. 量子系统为计算增加了什么结构？
3. 抽象计算怎样转化为物理处理器要求？
4. 通用控制为什么仍不等于可靠、可扩展的计算？
5. 在固定任务下，为什么考虑中性原子？
6. 在中性原子中，何时选择 $^{171}\mathrm{Yb}$？
7. 能级、控制和相互作用怎样形成可验收的量子门？
8. 物理故障怎样经过纠错周期成为逻辑错误和可信结果成本？

网站不是研究记录系统、实验放行工作站或自动改写论文的工具。保留论文追溯、术语索引和教学交互；移除主学习路径中的“研究交付”“工作站”“下一项研究任务”“放行合同”等措辞。

每个知识单元使用同一数据结构和视觉顺序：

## 5. 第一域：计算与物理实现

### 5.1 什么是计算

从任务定义开始：输入集合 $X$、输出集合 $Y$、规则或关系 $f$、允许误差和验收程序。确定性计算写作 $y=f(x)$；概率算法则规定条件分布 $p(y|x)$ 和成功判据。此处不引入量子态、POVM、泄漏或门保真度。

核心结论：计算不是某种材料或能级，而是一套可重复执行并可独立验证的输入—输出关系。物理机器必须提供编码、变换和读出接口。

### 5.2 量子计算改变了什么

依次引入：

- 密度算符表示可制备的量子状态；
- 张量积描述复合系统；
- 量子通道描述允许的物理演化；
- POVM 把量子状态映射为经典结果分布。

Born 规则放在测量步骤：

$$
p(m|\rho)=\operatorname{Tr}(E_m\rho),\qquad E_m\succeq0,\quad\sum_mE_m=\mathbb I.
$$

正文定义 $\rho$、$E_m$ 和 $m$，说明它回答“怎样从量子态得到可核验的经典记录”，不用于说明量子门怎样实现。叠加和纠缠用于解释状态空间与复合结构，不宣称它们自动带来计算加速。

### 5.3 DiVincenzo 五项准则

五项准则是物理处理器完整性检查，不是全文目录、性能排行榜或容错阈值。每项采用同一四联结构：

| 准则 | 回答的问题 | 缺失时的失败 | 中性原子证据 |
| --- | --- | --- | --- |
| 可扩展且可表征的量子比特 | 信息编码在哪里，非计算态是什么 | 规模增加后状态定义、串扰或损失口径失效 | 站点分辨谱、计算子空间、泄漏/真空分类 |
| 初始化到基准态 | 如何得到已知输入 | 输出混入未知初态误差 | 装载、重排、光抽运、复位混淆矩阵 |
| 相干时间长于相关操作时间 | 可执行多深的控制序列 | 相位信息在算法完成前丢失 | 同一运行条件下的门、移动、等待和测量相干 |
| 通用门集 | 是否能逼近任意目标电路 | 只能完成受限动力学或经典可模拟子集 | 任意单比特控制与至少一种经过相位验证的纠缠门 |
| 指定量子比特测量 | 如何得到局域经典结果 | 无法提取答案或综合征 | 站点分辨读出、SPAM、损失、泄漏和回作用 |

若展示 $T_{\mathrm{coh}}/t_{\mathrm{op}}$，必须标明它只是可执行操作深度的粗略尺度，不是门保真度、算法深度或容错阈值。

### 5.4 通用、纠错与容错

正文依次区分：

- 通用性：任意单比特控制加一种纠缠门可构成通用门集；
- 量子纠错：把逻辑信息编码到更大希尔伯特空间，通过综合征识别可纠正故障；
- 容错：电路结构限制单个故障的传播，使纠错过程本身不会破坏编码信息；
- 可扩展容错：在固定周期通道和调度下，增加码距持续降低逻辑错误率。

删除孤立的 $U_{AB}\ne U_A\otimes U_B$ 展示。若需要说明纠缠门，使用受控相位对计算基的作用及可测条件相位，而不是把“传播子不可分解”当作完整门验收条件。

## 6. 第二域：中性原子与 171Yb

### 6.1 为什么考虑中性原子

先固定比较任务：逻辑量子比特数、逻辑电路深度、目标失败概率、允许墙钟时间、量子码和损失处理口径。平台对比仅使用以下决策维度：

- 静置时的存储误差；
- 纠缠连接与路由方式；
- 可并行控制密度；
- 一次综合征周期时间；
- 测量、复位和损失记录；
- 控制、制冷、维护和失败重试成本。

中性原子的核心交换为：天然相同的原子和可重构几何降低器件制造与固定布线约束，基态弱相互作用与 Rydberg 强相互作用提供“存储时隔离、门期间耦合”；代价进入随机装载、运动、光学均匀性、Rydberg 衰变、原子损失和周期时延。

不得给出脱离任务的“最佳平台”结论。

### 6.2 为什么选择 171Yb

Rb、Cs、Sr 与 Yb 按同一功能链比较：冷却与装载、存储编码、单比特控制、Rydberg 纠缠、局域读出、复位、故障标记。比较表中的每一项必须是可验证的接口差异，不使用“成熟”“先进”“潜力大”等无条件评价词。

$^{171}\mathrm{Yb}$ 的核心论证为：$I=1/2$ 核自旋提供最小二能级核自旋空间；两价电子结构提供基态、亚稳钟态、窄线冷却态和 Rydberg 态等不同流形，使存储、门和读出可以分配到不同物理接口。该结构增加激光波长、偏振、磁场和流形映射的标定负担。

### 6.3 擦除转换的条件性优势

教学图固定展示以下信息流：

```text
Rydberg 门故障
-> 衰变到计算子空间内或外
-> 流形选择检测
-> 真阳性、漏报、误报与检测回作用
-> 带位置标签的周期记录
-> 解码器
-> 逻辑错误率
```

擦除是“位置已知的错误”，不是错误已经消失。Yb 优势只有在实测标签混淆矩阵、检测时延、旁观原子回作用、残余 Pauli 故障、相关故障和补原子代价全部进入周期模拟后仍降低逻辑错误率时成立。

理论预测、2023 年门与中途检测实验、2026 年逻辑比特实验必须分开表述，不把理论阈值、物理门结果和逻辑码实验混成同一证据。

## 7. 第三域：能级、门与测量

### 7.1 能级图

每条跃迁的详情固定回答：初态和末态、控制量、实验硬件、可观测量、主要副作用、来源。图中可见文字使用英文，页面说明随中英文切换。

### 7.2 从哈密顿量到条件相位

推导顺序固定为：

1. 旋转框架下单原子有效哈密顿量；
2. 两原子 $|rr\rangle$ 相互作用能移；
3. 阻塞条件与残余双激发；
4. 各计算基态积累的相位；
5. 解激发回计算子空间后的受控相位门。

每个公式定义状态空间、角频率单位、失谐符号、旋转波近似和忽略项。范德瓦耳斯近似必须注明 Förster 共振、角向耦合、多通道和运动平均的适用边界。

### 7.3 交互模型

控制量限制为 $\Omega$、$V$、$\Delta$、脉冲时间、Rydberg 寿命和温度。每次调整必须同时更新：

- 状态布居轨迹；
- 最大 $P_{rr}$；
- 条件相位及相位误差；
- Rydberg 时间积分与衰变暴露；
- Doppler 失谐尺度；
- 明确列项的教学误差预算。

删除综合“保真度代理”。只有在给出状态集合、目标门、损失口径和平均定义后，才显示计算得到的保真度；否则显示基础可观测量和误差项。界面需标注这是教学模型，不是特定装置的校准或实验预测。

## 8. 第四域：实验周期

实验页面按原子经历的时间顺序组织：源与真空、减速与冷却、光镊装载、成像与重排、态制备、单/双比特门、中途检测、解码反馈、复位与补原子。

每一步只回答：原子状态怎样变化、哪种装置施加哪一物理量、测量信号是什么、失败怎样进入下一周期。保留实验图版热点，但热点说明不得遮挡主图，移动端改为图下详情。删除仪器采购或研究工作站式语言。

## 9. 第五域：容错、规模与成本

错误层级固定为：

```text
物理耦合和噪声
-> 门级条件通道
-> 带时空相关和经典标签的周期通道
-> 综合征与擦除记录
-> 解码恢复或 Pauli frame
-> 每周期/逻辑门错误率
-> 码距缩放
-> 完成算法的资源与时间
```

教学资源估算器必须显示公式来源、独立噪声假设、阈值参数、未包含项目和单位。不得把表面码式经验缩放当成特定 Yb 装置预测。成本统一为同一问题、精度与成功标准下的一次可信结果成本，并包含失败重试与可用率。
