# 布洛赫球及其在中性原子量子计算中的应用

## 摘要

布洛赫球（Bloch sphere）是量子计算中最直观且最强大的可视化工具之一，它将一个二能级量子系统的所有可能纯态映射到单位球面上。在中性原子量子计算中，布洛赫球不仅是理解和设计单量子比特门的基础，更是分析和优化里德堡(Rydberg)两量子比特门（如CZ门）过程中，追踪量子态演化轨迹的核心手段。本笔记系统整理布洛赫球的数学基础、几何意义，以及它在中性原子量子计算，特别是里德堡阻塞门设计中的关键应用。

---

## 第一部分：布洛赫球的数学基础

### 1.1 二能级系统的态矢量

一个二能级量子系统的任意纯态（pure state）可以表示为基态 $|0\rangle$ 和 $|1\rangle$ 的叠加：

$$
|\psi\rangle = \alpha |0\rangle + \beta |1\rangle, \quad \alpha, \beta \in \mathbb{C}, \quad |\alpha|^2 + |\beta|^2 = 1
$$

忽略全局相位后，可以将 $\alpha$ 和 $\beta$ 参数化为：

$$
|\psi\rangle = \cos\frac{\theta}{2} |0\rangle + e^{i\phi} \sin\frac{\theta}{2} |1\rangle
$$

其中：
- **$\theta$**（极角，$0 \le \theta \le \pi$）：决定了在 $|0\rangle$ 和 $|1\rangle$ 之间的布居数分配。
- **$\phi$**（方位角，$0 \le \phi < 2\pi$）：决定了两个基态之间的**相对相位**。

### 1.2 布洛赫球的几何表示

布洛赫球是一个单位球面。上述参数 $(\theta, \phi)$ 恰好是球坐标，因此可以将任意纯态 $|\psi\rangle$ 唯一地映射到球面上的一个点，其笛卡尔坐标为：

$$
\begin{aligned}
x &= \sin\theta \cos\phi = 2\,\text{Re}(\alpha^* \beta) \\
y &= \sin\theta \sin\phi = 2\,\text{Im}(\alpha^* \beta) \\
z &= \cos\theta = |\alpha|^2 - |\beta|^2
\end{aligned}
$$

这个三维矢量 $\mathbf{r} = (x, y, z)$ 被称为**布洛赫矢量**。

### 1.3 球面上的关键点
| 量子态 | 布洛赫矢量坐标 | 几何位置 |
| :--- | :--- | :--- |
| $\vert0\rangle$ | $(0, 0, 1)$ | 北极 |
| $\vert1\rangle$ | $(0, 0, -1)$ | 南极 |
| $\frac{1}{\sqrt{2}}(\vert0\rangle + \vert1\rangle)$ | $(1, 0, 0)$ | +X轴 |
| $\frac{1}{\sqrt{2}}(\vert0\rangle - \vert1\rangle)$ | $(-1, 0, 0)$ | -X轴 |
| $\frac{1}{\sqrt{2}}(\vert0\rangle + i\vert1\rangle)$ | $(0, 1, 0)$ | +Y轴 |
| $\frac{1}{\sqrt{2}}(\vert0\rangle - i\vert1\rangle)$ | $(0, -1, 0)$ | -Y轴 |
### 1.4 布洛赫球的重要性质

- **纯态与球面**：所有纯态都位于球面上，即 $x^2 + y^2 + z^2 = 1$。
- **混合态与球内**：混合态（如热态、退相干态）位于球体内部，其布洛赫矢量长度小于1。
- **正交性**：球面上对径点（如 $|0\rangle$ 和 $|1\rangle$）对应的量子态彼此正交。
- **单比特门与旋转**：任何单比特量子门（幺正变换）都对应于布洛赫球上的一个**刚性旋转**。例如：
    - $X$ 门：绕 X 轴旋转 $\pi$。
    - $Z$ 门：绕 Z 轴旋转 $\pi$。
    - $R_x(\theta)$：绕 X 轴旋转角度 $\theta$。

## 第二部分：中性原子量子计算中的布洛赫球

### 2.1 中性原子量子比特的实现

在中性原子量子计算中，量子比特通常编码在原子的两个长寿命能级上。常见的编码方案包括：

- **基态超精细编码**：利用基态的两个超精细能级（如 $^{87}\text{Rb}$ 的 $|F=1,m_F=0\rangle$ 和 $|F=2,m_F=0\rangle$）。
- **里德堡编码**：利用基态（或亚稳态）与里德堡态（$|r\rangle$）构成二能级系统。

### 2.2 单量子比特操控与布洛赫球

在中性原子系统中，单量子比特门通常通过**微波场**（驱动超精细跃迁）或**激光场**（驱动光学跃迁）来实现。这些驱动场会产生一个等效的旋转场，使得布洛赫矢量绕着某个轴旋转。

**拉比振荡（Rabi Oscillation）**：在共振驱动下，布洛赫矢量绕着赤道面上的一个轴（通常为 X 或 Y 轴）匀速旋转。旋转角速度为拉比频率 $\Omega$，在时间 $t$ 后，量子态获得的旋转角为 $\theta = \Omega t$。

### 2.3 里德堡阻塞与两量子比特门

中性原子量子计算的一个核心优势是利用**里德堡阻塞效应（Rydberg Blockade）** 实现高保真度的两量子比特门。

- **里德堡阻塞**：当两个原子被激发到里德堡态时，它们之间会产生极强的范德瓦尔斯相互作用，使得双激发态 $|rr\rangle$ 的能量被大幅上移，从而被禁止。
- **两量子比特门设计**：在阻塞条件下，系统的动力学被限制在几个特定的子空间内。例如，对于初态 $|01\rangle$，系统在 $\{|01\rangle, |0r\rangle\}$ 这个二能级子空间中演化；对于初态 $|11\rangle$，系统则在 $\{|11\rangle, (|1r\rangle+|r1\rangle)/\sqrt{2}\}$ 这个二能级子空间中演化，且耦合强度增强为 $\sqrt{2}\Omega$。

## 第三部分：布洛赫球轨迹在门设计中的应用

在优化和验证里德堡门的过程中，追踪量子态在布洛赫球上的**演化轨迹**是至关重要的。

### 3.1 为什么需要观察轨迹？

- **验证门操作的保真度**：通过观察轨迹是否按设计路径演化并最终回到起点（或到达目标态），可以验证脉冲序列的正确性。
- **诊断误差来源**：轨迹的形状对实验参数（如失谐、拉比频率误差）非常敏感。分析轨迹偏离理想路径的程度，可以帮助定位误差来源。
- **可视化“闭合路径”与几何相位**：在非绝热和乐量子计算（NHQC）中，量子态的演化路径在布洛赫球上形成一个闭合回路。回路所包围的立体角（Solid Angle）正比于获得的几何相位（Berry Phase），而几何相位是实现容错门的关键。

### 3.2 二能级子空间的布洛赫球表示

如前所述，在里德堡阻塞门中，不同初态的演化被限制在不同的二能级子空间中。因此，这些子空间的动力学可以分别投影到各自的布洛赫球上进行可视化。

对于初态 $|01\rangle$，其演化发生在 $\{|01\rangle, |0r\rangle\}$ 子空间。定义该子空间的“基态”为 $|0\rangle_{\text{eff}} = |01\rangle$，“激发态”为 $|1\rangle_{\text{eff}} = |0r\rangle$。那么，初态 $|01\rangle$ 就对应这个有效布洛赫球的北极。激光脉冲会使布洛赫矢量在该球面上运动，其轨迹反映了布居数在 $|01\rangle$ 和 $|0r\rangle$ 之间的振荡以及相对相位的变化。最终，为了完成门操作，轨迹必须回到北极，即 $|0r\rangle$ 态的布居数为零。

### 3.3 实例：代码中的布洛赫球轨迹绘制

在提供的代码中，`bloch_from_two_level` 函数正是将二能级子空间的态矢量 $(a, b)$ 映射到布洛赫球坐标 $(x, y, z)$：

```python
def bloch_from_two_level(states):
    a = states[:, 0]
    b = states[:, 1]
    return (
        2*np.real(np.conj(a)*b),  # x 坐标
        2*np.imag(np.conj(a)*b),  # y 坐标
        np.abs(a)**2 - np.abs(b)**2,  # z 坐标
    )
```

这个函数被用于绘制两个关键子空间的轨迹：
- **`traj01_eff`**: 从 $|01\rangle$ 出发，在 $\{|01\rangle, |0r\rangle\}$ 子空间中的演化轨迹（橙色）。
- **`traj11_eff`**: 从 $|11\rangle$ 出发，在 $\{|11\rangle, |W\rangle\}$ 子空间中的演化轨迹（灰色）。

通过绘制这些轨迹，可以直观地验证AR（幅度鲁棒）脉冲是否实现了设计目标：两个轨迹都从北极出发，在球面上划出特定的路径，并在门结束时**准确回到北极**，从而保证 $|01\rangle$ 和 $|11\rangle$ 态只获得相位，没有布居数泄漏。

## 第四部分：总结

布洛赫球是理解和设计量子计算中单比特门及两比特门动力学的核心工具。在中性原子量子计算中，它具体应用于：

1.  **单比特门设计**：将微波或激光脉冲映射为布洛赫球上的旋转。
2.  **里德堡门设计**：将复杂的多能级系统分解为若干独立的二能级子空间，并在各自的布洛赫球上追踪演化轨迹，以验证和优化脉冲序列。
3.  **误差诊断**：通过分析实验测得的轨迹与理论轨迹的偏差，定位噪声和系统误差的来源。
4.  **几何量子计算**：利用布洛赫球上的闭合路径来设计和实现容错的几何门。

掌握布洛赫球的表示方法，是深入理解中性原子量子计算，特别是里德堡量子门物理机制的必要基础。
