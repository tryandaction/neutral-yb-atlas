# Website Application

This directory contains the React + TypeScript application for [Neutral Yb Atlas](https://neutral-yb-atlas.pages.dev/).

The public experience is a learning atlas: interactive diagrams and models explain the material, while the reading section links concepts to original papers and authoritative databases. It is not a research workspace and does not collect browser-side notes, snapshots or editable article copies. The footer contribution command opens a GitHub Issue form; only reviewed Pull Requests can change published content.

## Development

```powershell
npm install
npm run dev
npm test -- --run
npm run lint
npm run build
```

For public content changes, source requirements and review expectations, see [../CONTRIBUTING.md](../CONTRIBUTING.md).
