# Image2 中文实验教学图生成指南

## 目标

图片必须直接解释实验对象，而不是套用“学习目标、关键参数、实验验收、常见问题、为什么重要”的通用课件模板。理想结果接近一张高质量论文总图：真实装置承担空间与材料信息，剖面和局部示意揭示不可见结构，文字紧贴对象给出数值、调节动作和观测信号。

## 图文组织

- 顶部只保留一个直接标题，控制在画布高度的 6–8%。
- 真实装置或真实仪器占主体面积的 60–70%。
- 用 2–4 个局部放大窗补充内部结构、光路、势阱、PSF、波形或数据处理。
- 使用 5–8 个紧凑说明卡；每张卡只回答一个对象，不做重复总结。
- 不设置整排等宽底栏，不使用目标、齿轮、剪贴板、问号、星形等课件图标。
- 颜色只编码真实物理通道，例如 399 nm 蓝、556 nm 绿、578 nm 红、约 302 nm 紫。

## 每条说明的内容

说明卡优先使用以下结构，按主题选择其中两至四项：

```text
对象或通道
数值 / 范围 / 公式
操作：先调什么，再调什么
观测：用什么仪器，看什么曲线或信号
判据：什么结果允许进入下一步
```

禁止只写“提高稳定性”“保证精度”“优化性能”“非常重要”等不可执行语句。

## 数值来源标记

所有数值必须属于以下四类之一，并在图中或提示词中明确身份：

1. `原子固有值`：例如 399 nm 线宽 `Γ/2π ≈ 29 MHz`、556 nm 线宽 `Γ/2π ≈ 182 kHz`。
2. `文献装置示例`：例如 `NA=0.6`、399 nm 快速成像 `20 μs / 0.986`、302 nm 链路 `50 mW → 6 mW / w₀=10 μm`。
3. `常见工程起点`：例如 UHV、烘烤或炉温范围；必须注明受材料、密封和结构限制。
4. `本机标定`：没有通用数值时，直接写测量变量、扫描方法和放行算法，不编造统一阈值。

不要把单篇论文的装置数据写成所有 Yb 系统的标准规格。

## 推荐生成设置

- 模型：Image2 / `gpt-image-2`
- 画布：`3840×2160`，16:9 横向
- 质量：high
- 格式：PNG，sRGB
- 一次生成一张；先选物理结构正确的版本，再逐区修正文字
- 图片内使用中文；波长、单位、公式和 MOT、RGA、AOM、EOM、FPGA、AWG、QEC 等标准缩写保持原样

## 参考图使用方式

生成时建议同时附上：

- `Image 1`：`Concept/Yb原子计算原理图-能级与光参数.png`，仅作为信息密度、直接说明、短引线和局部卡片的参考；不得复制其能级内容。
- `Image 2`：第一版“镱原子源与超高真空链路”生成图，作为反例和材质参考；保留真实金属设备质感，删除学习目标框、四栏底板、装饰图标和空泛语言。

## 逐图验收

拒绝并重新生成或局部修正以下结果：

- 出现学习目标、为什么重要、通用四栏底板或大装饰图标；
- 文字只有原则，没有数值、操作或观测信号；
- 同一句含义在标题、编号卡和底部总结中重复；
- 主设备只有外观，没有提示词要求的内部剖面或局部示意；
- 装置真实但关键路径、方向、接口或测量点不清楚；
- 文献示例未标身份，或工程起点被写成绝对标准；
- 中文错字、伪中文、公式错误、单位错误或额外标签；
- 说明卡遮挡器件，或引线无法唯一确定目标；
- 出现厂商、logo、水印、营销口号或科幻装置。

原图缩放到 50% 时，标题、数字、公式和操作短句仍应可辨。网页热点只展开更详细的双语说明，不重复图片内已有文字。

```text
Constraints: 只替换指定文字；器件、剖面、光路、引线、其他文字、颜色、布局和画布尺寸完全不变；不得新增内容
```
