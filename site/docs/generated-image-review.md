## 发布门槛

1. 波长、能级、单位和跃迁符号逐项核对，不接受形近字符或同音替换。
2. 不出现无来源的功率、光腰、工作距离、消光比、延迟、shot ID、frame ID 或时间戳。
3. 文献值必须和“本机复标”要求同时出现；示意曲线必须标注“示意”。
4. 图内文字用于直接说明结构、操作、记录量和边界，不加入学习目标、宣传语或重复总结。
5. 公开页面只引用状态为“发布”“遮罩后发布”或“校正说明后发布”的资产。
